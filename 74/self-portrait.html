P5.JS SELF-PORTRAIT






I used an application called p5.js and coded with javascript a self-portrait of myself! So this image you're looking at right now isn't a picture that was drawn digitally, it was all typed out through code that is being run on this page to create shapes and formatted in a way that these shapes layered correctly in order to create this self-portrait of myself!
