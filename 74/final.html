Retro Re:Imagined Final



Retro Re:Imagined is my final project to create 3d models of iconic video game mascots and characters in their original debut forms. This is in order to celebrate the culture of video game history that has left a big impact on many people from back in the day and still even now. My aim is to reintroduce the old and combine them in the new in a sense by taking the old 2d sprites and extruding them into a new dimension that is 3d.

After creating 3d objects in minecraft, a program called mineways is used to extract the code of the object directly from our minecraft world's save file where the object was crafted. The extracted code is imported and creates a 3d model, which the file can be exported onto a program that lets you 3d print it.
